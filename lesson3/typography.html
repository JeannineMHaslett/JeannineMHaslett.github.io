Typography/Font Choices

Header

The main heading will be "Bree Serif". This font was chosen because serif font is standard for professional websites, but this serif font has rounded serifs, making it appear a little more stylish.

Navigation

The nagiation labels will be "Poppins", which is a sans-serif font. It was chosen because it is open and round with the lower-case a formed the way it is in hand print, which makes it easy to see and read. This type of font comes in several types--medium, bold and italic.

Main headings

Section headings will be "Bree Serif". This font was chosen because it matches the main heading font and makes new paragraph or section identification easier. It makes the page cohesive.

Secondary Headings

The secondary headings will be "Poppins". This font was chosen because at smaller sizes, "Bree Serif" becomes more difficult to read. Using a different typeface also distinguishes the difference between headings and subheadings.
